# **App Name**: Manual Master

## Core Features:

- Manual Display: Display manuals in a clean, readable format.
- Search Functionality: Allow users to quickly search through manuals.
- Category Filter: Enable users to filter manuals by category.
- AI Summarization: Summarize lengthy manuals using generative AI, offering quick overviews; uses the LLM as a tool to decide what content is worth summarizing.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for trust and clarity.
- Background color: Light gray (#F5F5F5) for easy readability.
- Accent color: Soft green (#81C784) for highlighting important sections.
- Body font: 'Inter', sans-serif, for clear and accessible text.
- Headline font: 'Space Grotesk', sans-serif, a slightly more stylized sans-serif suitable for drawing the eye to headings and titles.
- Use simple, professional icons to represent different manual categories.
- Clean and structured layout to ensure readability and ease of navigation.
